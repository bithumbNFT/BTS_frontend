# ✨ BTS (Bithum NTF SNS) 💰

## 👉 프로젝트 소개


- 프로젝트명은 **Bithum의 NFT와 SNS 서비스를 결합하여 BTS로 지었습니다.** 다양한 작가들의 그림과 아트를 NFT 코인 경매로 만나볼 수 있으며 빗썸 코인 유저들과 열린 소통을 할 수 있도록 SNS 커뮤니티도 활성화 시켰습니다.
